# Job

## Job Schema

- title string
- slug string
- description string
- email string
- address string
- company string
- industry enum
  - Business
  - Information Technology
  - Banking
  - Education/Training
  - Other
- degrees
  - BSC
  - MSC
  - PHD
- positions number
- experience enum
  - No Experience
  - 1 Year - 2 Years
  - 2 Years - 5 Years
  - 5 Years+
- salary number
- postingDate date
- lastDate date
- applicantsApplied number
